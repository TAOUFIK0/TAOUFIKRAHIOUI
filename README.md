# Mon Portfolio Personnel

Portfolio professionnel présentant mes compétences, projets et parcours en ingénierie.

## 🚀 Fonctionnalités

- **Design Responsive** : Compatible avec tous les appareils
- **Navigation Fluide** : Scroll smooth entre les sections
- **Téléchargement CV** : Bouton de téléchargement direct
- **Galerie de Projets** : Présentation détaillée de mes réalisations
- **Timeline Formation** : Parcours académique et professionnel
- **Formulaire de Contact** : Contact direct via le site

## 📁 Structure du Projet

```
portfolio/
├── index.html          # Page principale
├── style.css           # Styles CSS
├── script.js           # JavaScript
├── assets/             # Ressources
│   ├── profile-photo.jpg
│   ├── CV_VotreNom.pdf
│   ├── project1.jpg
│   └── project2.jpg
└── README.md
```

## 🛠️ Technologies Utilisées

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (Vanilla)
- Font Awesome (Icônes)

## 📋 À Personnaliser

1. **Informations personnelles** dans `index.html`
2. **Photo de profil** dans `assets/profile-photo.jpg`
3. **CV** dans `assets/CV_VotreNom.pdf`
4. **Images de projets** dans `assets/`
5. **Couleurs** dans `style.css` (variables CSS)

## 🚀 Déploiement

### GitHub Pages
1. Créez un repository GitHub
2. Uploadez tous les fichiers
3. Activez GitHub Pages dans les paramètres
4. Votre portfolio sera accessible via `https://username.github.io/repository-name`

### Netlify
1. Connectez votre repository GitHub
2. Déployez automatiquement à chaque commit
